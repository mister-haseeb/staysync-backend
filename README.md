# StaySync – Event / Meetup Planner

## 📌 Project Overview

StaySync is a full-stack MERN application where users can create and manage events. Other users can RSVP to events to show their participation.

## 🚀 Features

* Create events (title, date, location)
* View all events
* RSVP to events
* User-friendly interface
* Full-stack MERN implementation

## 🛠 Tech Stack

* MongoDB
* Express.js
* React.js
* Node.js

## 📁 Project Structure

* `/backend` → API and database logic
* `/frontend` → React user interface

## 📦 Installation

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
